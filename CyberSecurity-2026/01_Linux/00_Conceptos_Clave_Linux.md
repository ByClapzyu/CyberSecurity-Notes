
# Terminal

Es la ventana o interfaz grafica que pretende ser el hardware fisico que se usaba antes para interactuar con las computadoras.

# Shell

Es la interfaz del usuario real que interactua con el nucleo (kernel) de linux. El terminal solo nos da acceso al shell

# Bash

**Born Again Shell**. Es el tipo de shell más común en Linux. Existen otras como zsh, fish o incluso pwsh.

# Simbolos del Prompt

- **$** Usuario normal
- **#**(Usuario Root) indica que se tiene todo el poder sobre el sistema
