
# vm_conf_red

- NAT (**Network Address Translation**): Permite



- **NAT (Network Address Translation):** Imagínalo como un router que le da salida a internet a tu servidor pero lo esconde del resto de tu casa. **¿Para qué sirve?** Para que el servidor pueda descargar actualizaciones de Windows y el rol de AD. Una vez que el laboratorio esté listo, solemos apagarla para un aislamiento total.
    
- **Red Interna (Internal Network):** Es como si compraras un switch físico, lo pusieras en una caja fuerte y solo conectaras tus máquinas virtuales ahí. **¿Para qué sirve?** Para que el tráfico de tu dominio (como cuando un cliente se loguea) no salga a tu red de Wi-Fi real. Es el "perímetro de seguridad" de tu laboratorio.
